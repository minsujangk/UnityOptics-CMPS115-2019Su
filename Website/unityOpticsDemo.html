Unity Optics

An Advertisement Tool for the Unity 3D Game Engine

<

Unity Optics is a plugin for the Unity3D Game Engine. It enables developers to add 3D-based advertisement tools to their in-game objects. Simply attach the plugin script to any 3D object and you can record all currently available telemetry options: Distance, time spent watching, whether they read the advertisement, and more!

Unity Optics Was Developed With:

Unity 3D

The Unity3D Game Engine in C#

Firebase

Google Firebase Storage and Realtime Database

Front-End

HTML5, CSS, and Javascript

Video Demo

In this video demo you can view the interactions added to 3D objects within your game. All of these actions are recorded and stored in Firebase Storage for later retrieval.
